Aprenda Posicionamento CSS

O posicionamento CSS é uma propriedade fundamental para controlar a localização de elementos na sua página web. Existem cinco valores principais para a propriedade `position`:

---

1. position: static; (Padrão)

Este é o valor padrão para todos os elementos HTML. Elementos com position: static; são renderizados na ordem normal do documento. As propriedades top, bottom, left e right não têm efeito.

Static Box 1
Static Box 2

Os boxes acima estão com position: static;. Observe que eles se comportam como blocos de texto normais, um após o outro, sem serem afetados por deslocamentos.

---

2. position: relative;

Um elemento com position: relative; é posicionado em relação à sua posição normal. Você pode usar as propriedades top, bottom, left e right para deslocá-lo de sua posição original. O espaço que ele ocuparia no layout normal permanece reservado.

Normal Box
Relative Box
Another Box

O "Relative Box" está deslocado 20px para baixo (top: 20px;) e 20px para a direita (left: 20px;) de sua posição original. O espaço "vazio" que ele deixou para trás ainda está presente.

---

3. position: absolute;

Um elemento com position: absolute; é posicionado em relação ao seu **ancestral posicionado mais próximo** (um elemento cujo position não é static). Se não houver um ancestral posicionado, ele é posicionado em relação ao documento (``). Elementos absolutos são removidos do fluxo normal do documento, o que significa que não ocupam espaço.

Base Box
Absolute Box

O "Absolute Box" está posicionado 50px de cima (top: 50px;) e 20px da direita (right: 20px;) **em relação ao seu container pai** (que tem position: relative;). Repare que ele não afeta o layout dos outros boxes.

---

4. position: fixed;

Um elemento com position: fixed; é posicionado em relação à **janela de visualização (viewport)**. Ele permanece na mesma posição mesmo quando a página é rolada. Isso é útil para cabeçalhos ou rodapés fixos.

Role a página para baixo para ver o efeito do "Fixed Box".





















Fixed Box

O "Fixed Box" está fixo no canto inferior direito da sua tela. Ele não se move quando você rola a página.

---

5. position: sticky;

Um elemento com position: sticky; é posicionado com base na posição de rolagem do usuário. Ele se comporta como `relative` até que uma determinada posição de rolagem seja atingida, momento em que ele se torna `fixed` em relação à janela de visualização.
